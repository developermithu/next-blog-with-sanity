## How To Use 

### Frontend
```
git https://github.com/developermithu/next-blog-with-sanity.git
cd next-blog-with-sanity
npm install
npm run dev
```

### Backend
```
cd sanity-backend
sanity start
```

### Plugins In Frontend 
- next-sanity
- portabletext/react
- sanity/image-url
- tailwindcss/typography
- react-syntax-highlighter
